Crypto API Automation

This project involved using CoinMarketCap's API to collect data on cryptocurrency and automating the data collection process. The collected data was then analyzed using Python's Pandas library to create DataFrames, which were used to generate visualizations using Seaborn and Matplotlib. The visualizations helped to identify trends and patterns in the cryptocurrency market. The project demonstrated the ability to use APIs, automate data collection, and apply data analysis techniques to extract insights from the data.
